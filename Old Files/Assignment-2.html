Part A - Cross Validation¶

Exercise A0¶

Describe (in your own words) the principle of k-fold cross validation (Section 2.2.1).

k-fold cross validation divides your training data set into k different parts(folds). You then iterate over the folds and for each fold you attempt to train the model on the other folds. The selected fold in the iteration is then used to test how good the model fits the data. The final training error of the model is the average error over all the iterations. This way we tend to avoid overfitting more, and get a more realistic measurement of the model

In [138]:
def CV(clf,X,Y,I,K):
    errors=0
    predictions=0

    #CV loop
    for k in range(0,K):
        ##Get sets
        X_test = np.array(X)[I==k, :]
        X_train = np.array(X)[I!=k, :]
        Y_test = np.array(Y)[I==k]
        Y_train = np.array(Y)[I!=k]

        ##Train the classifier
        clf2 = clf.fit(X_train, Y_train)

        ##Attempt prediction of the test set
        Y_hat=clf2.predict(X_test)

        ##Increment error and prediction count values
        for i in range(0,len(Y_hat)):
            predictions = predictions + 1
            if not(Y_hat[i]==Y_test[i]): errors = errors + 1
    return [errors,predictions]

def Train_E(clf,X,Y):
    clf2 = clf.fit(X, Y)

"""This method returns the left side of a link"""
def link_left_side(link):
    return link.split('|')[0]

"""This methods returns the internal wiki links"""
def get_wiki_links(text):
    links = re.findall("\[\[([^\]]*)\]\]",text[0]['*'])
    clean_links = []
    for link in links:
        clean_links.append(link_left_side(link))
    return clean_links

def cleanPage(page):
    string = ""
    for s in re.split(' |{|}|<!--|-->|\[|\]|\||<.*>|:|\*|\(|\)|,|#......|\n|\'|\.|=|\"|\/|\_|\%',page['revisions'][0]['*'])[:]:
        singles = re.findall('.',s);
        if (s != '' and s != ' ' and s != '\n' and s != '\n\n' and s != u'' and s!='=' and s!='\.' and not(len(singles)==1)):
            #print s
            string += s+' '
    return string

def test_stub(page):
    search = re.findall('\[\[.*\]\]*(.*)',page['revisions'][0]['*'])
    result = False
    for word in search:
        if word != '':
            result = True
    return result
